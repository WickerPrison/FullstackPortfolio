Pixel Patchwork

Summary

This website allows users to place pixels one at a time to collaboratively make pixel art. Users can pay a one time fee to unlock a premium account to access faster pixel placement. It also features a live chat that allows users to coordinate their pixel art efforts. It was made by a group of 4 people.

Technologies Used

Made using React, Material UI, MongoDB, Mongoose, Socket.io, Apollo, GraphQL, and Stripe.

My Contributions

I was the team member primarily responsible for the back end. I did the majority of the work involving Apollo and GraphQL on the front and back end. I also contributed heavily to programming the canvas to display the pixels once retrieved from the back end in a way that was performant. I also implemented payment processing and the premium account features.

Features

Collaborative Canvas

Once logged in, users can change the color of any pixel on the canvas. This change is stored using MongoDB along with other information such as the user who made the change and the time the change was made. Other users who view the website can see this information by hovering over the pixel with their mouse.

Live Chat

Once logged in users will see a chat window in the bottom left corner of the screen. We implemented this chat feature using web sockets so that users can communicate in real time to coordinate their collaborative pixel art efforts.

Payment Processing

We also implemented payment processing using Stripe to allow users to unlock a premium account. By default, users may only place one pixel every 30 seconds, but by paying a small one-time fee, they can unlock a premium account that allows them to place a pixel every 3 seconds instead.
